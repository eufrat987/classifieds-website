# classifieds-website
A simple spring boot REST service that manages publications.


### Features
- Base CRUD operation support
- Auto-increment views on GET

### Technologies used
- Java 17+
- Spring Boot 3+
- Spring Data JPA
- H2 in-memory database
- Gradle 9.0.0

### Example requests 
/postman - Postman test API collection

## How to run
```bash
./gradlew bootRun
```
